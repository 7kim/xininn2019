6.you-get

官方地址：官网地址

GitHub仓库地址：客栈仓库

GitHub仓库中文说明：仓库软件中文说明

python下载地址：下载python

FFmpeg下载地址：下载FFmpeg

一小小哒命令行程序，提供便利的方式来下载网络上的媒体信息

必要前提条件：python 3.2或以上、ffmpeg 1.0或以上

命令行安装：pip3 install you-get

测试python有没有安装成功命令：python -V

测试you-get有没有安装成功命令：you-get -V

测试FFmpeg有没有安装成功命令：ffmpeg -version

升级命令：pip3 install --upgrade you-get

查看可用画质和格式的视频命令（标有DEFAULT为默认画质）：you-get -i '这里输入视频地址'

可以试用快捷键 ctrl + c 暂停下载

使用 --output-dir/-o 设定路径, --output-filename/-O 设定输出文件名

如 ： you-get -o ~/Videos -O wen '视频地址'

使用 --http-proxy/-x 为 you-get 设置HTTP代理:

如：you-get -x 127.0.0.1:8087 '视频地址'

7.annie

GitHub地址：项目地址

可以把你下载好的annie.exe文件放到 C:\Windows\System32 里面去

下载完后，要把ffmpeg里面的bin文件夹路径放到系统的环境变量里面

所有可用命令：annie -h

如果输入命令 ffmpeg -version 没有报错说明你成功了哈

食用方法：

先用命令 annie -i 粘贴你的视频地址

注意：如果URL包含特殊字符，请用引号将其括起来

然后回车，然后下面会列出可以下载的视频资源

接下来就是下载视频了，在列出来的列表中选中你需要下载的格式

批量下载： annie -i -p 视频地址

可以使用-x选项指定HTTP代理

annie -x http://127.0.0.1:7777 -i 视频地址

可以使用-s选项指定socks5代理

annie -s 127.0.0.1:1080 -i 视频地址
